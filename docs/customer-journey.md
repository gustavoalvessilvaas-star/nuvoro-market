# Customer Journey

1. Customer arrives from content, ads, search or email.
2. Home explains smart everyday essentials.
3. Product page explains problem, solution, offer, shipping and support.
4. Add to Cart opens mini-cart.
5. Buy Now sends the product to checkout.
6. Checkout collects minimal shipping/contact details.
7. Stripe handles payment.
8. Webhook updates order status.
9. Resend sends confirmation when configured.
10. Customer tracks order with order ID and email.
11. Support form handles questions or issues.
