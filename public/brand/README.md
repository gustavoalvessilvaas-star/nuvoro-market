# Nuvoro Market Brand Assets

Upload the final approved logo files here:

- `nuvoro-logo.svg` for the responsive header and footer logo.
- `nuvoro-logo.png` as a fallback raster logo.
- `nuvoro-icon.png` for favicon/app icon use.
- `nuvoro-og.png` for social sharing/Open Graph previews.

The app currently uses a polished text fallback if `nuvoro-logo.svg` and `nuvoro-logo.png` are missing. Do not use unapproved or copyrighted brand artwork.
